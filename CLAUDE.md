## Architecture

**Stack:** React 19, TanStack Router (file-based), TanStack Query, Tailwind CSS v4, Vite 7, TypeScript (strict)

### Routing

TanStack Router with file-based routing — `src/routes/` maps directly to URLs. **Never edit `src/routeTree.gen.ts`** — it's auto-generated by Vite on dev/build. Root layout (`__root.tsx`) wraps all routes with Header/Footer, and conditionally shows an `UnderConstruction` page based on the `VITE_CONSTRUCTION_DISABLED` env var.

Dynamic project routes use `$projectKey` — the key maps to project pathnames registered in `src/constants/project.ts`.

### Project Data System

Projects live in `src/projects/` as class instances implementing `ProjectProps`. They're registered in `src/constants/project.ts` in a `projectsByType` map keyed by URL pathname. To add a project: create a file in `src/projects/`, then add it to the registry in `src/constants/project.ts`.

### Technology / Badge System

`src/types/TechTypes.ts` defines a `techType` enum with 58+ technologies. Each maps to a Tailwind color scheme for badge rendering. Multiple techs can share a color via the compressed `techColorMap`.

### UI Components

- `src/components/ui/` — primitives: `PageHeader`, `PageDescription`, `TextLink`, `ExternalSite`, plus CVA-based `badge` and `card`
- `@abumble/design-system` — private npm package providing `cn` utility and additional components
- Icons via `lucide-react`
